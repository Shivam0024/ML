# Graph-Based Face Image Reconstruction Using GNNs

This repository contains code and resources for reconstructing face images using Graph Neural Networks (GNNs). The project processes the Morph2 dataset by constructing facial graphs and reconstructing images from embeddings generated by a GNN encoder.
